# Bark Studio® Website

A recreation of the "Bark Studio®" website - Where Paws Meet Pixels. This is a creative agency website with a fun dog theme, featuring smooth scrolling animations, responsive design, and interactive elements.

## Features

- **Responsive Design**: Fluid typography and layouts that adapt to all screen sizes
- **Smooth Scrolling**: Powered by Lenis for buttery smooth scroll experience
- **GSAP Animations**: Scroll-triggered animations and interactive effects
- **Video Background**: Hero section with lazy-loaded video
- **Interactive Navigation**: Dropdown menu with smooth scroll navigation
- **Team Profiles**: Animated profile cards with hover effects
- **Services Section**: Alternating layout with paw print decorations
- **Contact Form**: Functional form with success/error messaging
- **Modern CSS**: CSS Grid, Flexbox, and modern styling techniques

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Grid, Flexbox, Custom Properties, Animations)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- ScrollTrigger (GSAP plugin)
- Lenis (Smooth scrolling library)
