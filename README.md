# Bark Studio® Website

This is a recreation of the Bark Studio creative agency website from Webflow, built with pure HTML, CSS, and JavaScript.

## 🐕 Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll-based animations and interactive elements
- **Menu Navigation**: Toggle menu with smooth scroll navigation
- **Hero Section**: Video background with animated intro
- **The Pack Section**: Team member showcase with hover effects
- **Services Section**: Interactive service list with paw animations
- **Contact Form**: Functional contact form with validation
- **Image Gallery**: Sticky scroll image gallery section

## 🚀 How to Use

1. **Open Locally**: Simply open `index.html` in your web browser
   ```bash
   open index.html
   ```
   or
   ```bash
   firefox index.html
   ```

2. **Serve with a Local Server** (recommended):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```
   Then visit `http://localhost:8000`

## 🎯 Interactive Features

- **Menu Toggle**: Click the paw icon to open/close navigation
- **Smooth Scrolling**: Click menu items to scroll to sections
- **Form Validation**: Contact form with success/error messages
- **Hover Effects**: 
  - Profile cards with pink borders
  - Service items with paw animations
  - Menu items with color changes
- **Scroll Animations**: Text and elements animate as you scroll
- **Parallax Effects**: Images move at different speeds while scrolling
